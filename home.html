HouseLook API

Find your next house using our API!

This API provides information about about houses in different cities and towns. Users can query the website for all the houses in a place. One can get information about single rooms, bed sitters, one bedrooms, and two bedrooms in given area. Reviews are also provided by previous house tenants to help make relocation decision.

Resources

The API currently have 6 major resources:

  • /places
  • /houses
  • /rooms
  • /owners
  • /users
  • /reviews

Guide to access resources

Places

METHOD ROUTE DESCRIPTION
GET /places Obtain all available places within the database
GET /places/1 Get information about a place by providing a valid place id
Invalid id will return empty dictionary
GET /places/1/houses Get a list of houses in place by providing a valid place id
GET /places/8/houses/vacant Get a list of vacant houses in a place by providing the valid place id
GET /places/8/rooms Get a list of all rooms in a place
GET /places/8/rooms/singles Get a list of all single rooms in the place
GET /places/8/rooms/bedsitters Get a list of all bedsitters in a given place
GET /places/8/rooms/onebedrooms Get a list of all onebedrooms in a place
GET /places/8/rooms/twobedrooms Obtain all two bedrooms in a place
POST /places Create a new place
Ensure name is not null
PUT /places Update a place
Ensure name is not null
DELETE /places/89 Delete a place by providing valid Id

Houses

METHOD ROUTE DESCRIPTION
GET /houses Get all houses
GET /houses/3 Get information about a house by providing a valid house id
Invalid id will return empty dictionary
GET /houses/vacant Get all vacant houses
GET /houses/3/rooms Get all vacant rooms in a house by providing house id
GET /houses/3/owners Get the contact information of the house owener
GET /houses/3/reviews Get all reviews of a given house id
POST /houses Create a new house
Ensure name and place_id is not null
PUT /houses/8 Update house information
Ensure name and place_id is not null
DELETE /houses/7 Delete a house
Ensure rooms related to the house are deleted

Rooms

METHOD ROUTE DESCRIPTION
GET /rooms Get all rooms
GET /rooms/4 Get a specific room of given id
GET /rooms/vacant Get all vacant rooms
GET /rooms/singles Get all single rooms
GET /rooms/bedsitters Get all bedsitter rooms
GET /rooms/onebedrooms Get all one bedrooms
GET /rooms/twobedrooms Get all two bedrooms
GET /rooms/4/reviews Get reviews of specific room
POST /rooms Create a new room
Ensure house_id and place_id not null
PUT /rooms/89 Update a new room
Ensure house_id and place_id not null
DELETE /rooms/89 Delete room
Enter valid room id

Reviews

METHOD ROUTE DESCRIPTION
GET /reviews Get all reviews
GET /reviews Get all reviews
GET /reviews/1 Get a review of a given id
POST /reviews Create a review
Ensure house, user, and room ids are not null
DELETE /reviews/89 Delete a specific review

Owners

METHOD ROUTE DESCRIPTION
GET /owners Get a list of all owners
GET /owners/3 Get owner of given id
POST /owners Create a new owner
Ensure name and house id is not null
DELETE /owners/id Delete owner of given id

Users

METHOD ROUTE DESCRIPTION
GET /users Get a list of all users
GET /users/2 Get user of given id
POST /users Create a new user
Ensure name is not null
DELETE /users/89 Delete a user of given id
